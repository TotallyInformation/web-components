Fires:
  • uib-theme-changer:construction - Document object event. evt.event:details contains the data
  • uib-theme-changer:connected - When an instance of the component is attached to the DOM. `evt.event:details` contains the details of the element.
  • uib-theme-changer:disconnected - When an instance of the component is removed from the DOM. `evt.event:details` contains the details of the element.
