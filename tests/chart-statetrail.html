Documentation.

Alpha: Experimental. Documentation HERE.

Show a list of states for one or more items over time. Each item is shown as a separate line. States are shown as coloured blocks along a horizontal timeline.

Each state block has a tooltip showing the state name, value, and the start and end times.

Clicking on a block will trigger a custom event and (if Node-RED/UIBUILDER is used) will send a message back to Node-RED.
